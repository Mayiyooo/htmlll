Apa yang Akan Diseret - ondragstart dan setData() Kemudian, tentukan apa yang akan terjadi saat elemen diseret. Dalam contoh di atas, ondragstartatribut elemen memanggil fungsi (dragstartHandler(ev)), yang menentukan data apa yang akan diseret. Metode ini dataTransfer.setData()menetapkan tipe data dan nilai data yang diseret: function dragstartHandler(ev) { ev.dataTransfer.setData("text", ev.target.id); } Dalam kasus ini, tipe datanya adalah "teks" dan nilainya adalah id elemen yang dapat diseret ("img1"). Tempat Menjatuhkan - ondragover Atribut ondragoverelemen
memanggil fungsi (dragoverHandler(ev)), yang menentukan di mana data yang diseret dapat dijatuhkan. Secara default, data/elemen tidak dapat dihapus di elemen lain. Untuk mengizinkan penghapusan, kita harus mencegah penanganan default elemen tersebut. Hal ini dilakukan dengan memanggil preventDefault()metode untuk acara ondragover: function dragoverHandler(ev) { ev.preventDefault(); } Lakukan Drop - ondrop Ketika data yang diseret dijatuhkan, peristiwa jatuh akan terjadi. Dalam contoh di atas, atribut ondrop dari elemen
memanggil fungsi, dropHandler(event): function dropHandler(ev) { ev.preventDefault(); const data = ev.dataTransfer.getData("text"); ev.target.appendChild(document.getElementById(data)); } Kode dijelaskan: Panggilan preventDefault()untuk mencegah penanganan data default browser (defaultnya adalah terbuka sebagai tautan saat dijatuhkan) Dapatkan data yang diseret dengan dataTransfer.getData()metode ini. Metode ini akan mengembalikan data apa pun yang telah diatur ke tipe yang sama dalam setData()metode ini.
