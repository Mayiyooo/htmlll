XML adalah bahasa markup di mana semua dokumen harus ditandai dengan benar (berformat "baik"). XHTML dikembangkan untuk membuat HTML lebih mudah diperluas dan fleksibel agar dapat digunakan dengan format data lain (seperti XML). Selain itu, peramban mengabaikan kesalahan pada halaman HTML, dan mencoba menampilkan situs web meskipun terdapat beberapa kesalahan dalam markupnya. Oleh karena itu, XHTML hadir dengan penanganan kesalahan yang jauh lebih ketat. Jika Anda ingin mempelajari XML, silakan baca Tutorial XML kami .
