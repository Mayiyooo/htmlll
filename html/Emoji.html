Emoji tampak seperti gambar atau ikon, tetapi sebenarnya tidak. Itu adalah huruf (karakter) dari set karakter UTF-8 (Unicode). UTF-8 mencakup hampir semua karakter dan simbol di dunia. Atribut charset HTML Untuk menampilkan halaman HTML dengan benar, peramban web harus mengetahui rangkaian karakter yang digunakan pada halaman tersebut. Hal ini ditentukan dalam tag: Jika tidak ditentukan, UTF-8 adalah set karakter default dalam HTML. Karakter UTF-8 Banyak karakter UTF-8 tidak dapat diketik pada papan ketik, tetapi selalu dapat ditampilkan menggunakan angka (disebut nomor entitas): A adalah 65 B adalah 66 C adalah 67 Contoh

I will display A B C

I will display A B C

Contoh Dijelaskan Elemen mendefinisikan set karakter. Karakter A, B, dan C ditampilkan dengan angka 65, 66, dan 67. Agar browser memahami bahwa Anda menampilkan karakter, Anda harus memulai nomor entitas dengan dan mengakhirinya dengan ; (titik koma).
