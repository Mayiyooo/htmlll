Atribut charset HTML Set karakter ditentukan dalam tag: Contoh Spesifikasi HTML5 mendorong pengembang web untuk menggunakan set karakter UTF-8. Standar ini mendefinisikan 128 karakter berbeda yang dapat digunakan di internet: Huruf Inggris (AZ) Angka (0-9) Karakter khusus seperti ! $ + - ( ) @ < >. Set karakter ini mendukung 256 kode karakter yang berbeda. HTML 4 juga mendukung UTF-8.
