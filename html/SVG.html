SVG (Grafik Vektor yang Dapat Diskalakan) SVG mendefinisikan grafik berbasis vektor dalam XML , yang dapat langsung disematkan di halaman HTML. Grafik SVG dapat diskalakan, dan tidak kehilangan kualitas jika diperbesar atau diubah ukurannya: SVG SVG didukung oleh semua peramban utama. Apa itu SVG? SVG adalah singkatan dari Scalable Vector Graphics SVG digunakan untuk mendefinisikan grafik berbasis vektor untuk Web SVG mendefinisikan grafik dalam format XML Setiap elemen dan atribut dalam file SVG dapat dianimasikan SVG adalah rekomendasi W3C SVG terintegrasi dengan standar lain, seperti CSS, DOM, XSL dan JavaScript Elemen IKLAN Persegi Panjang SVG Contoh Persegi Panjang SVG dengan Opacity dan Sudut Membulat Contoh Bintang SVG Contoh SVG Gradien Elips dan Teks SVG Contoh Perbedaan Antara SVG dan Canvas SVG adalah bahasa untuk mendeskripsikan grafik 2D dalam XML, sementara Canvas menggambar grafik 2D secara langsung (dengan JavaScript). SVG berbasis XML, yang berarti setiap elemen tersedia di dalam SVG DOM. Anda dapat menambahkan pengendali peristiwa JavaScript ke grafik SVG. Dalam SVG, setiap bentuk yang digambar akan diingat sebagai sebuah objek. Jika atribut objek SVG diubah, peramban dapat merender ulang bentuk tersebut secara otomatis. Kanvas dirender piksel demi piksel. Di kanvas, setelah grafik digambar, gambar tersebut akan dilupakan oleh peramban. Jika posisinya harus diubah, seluruh adegan perlu digambar ulang, termasuk objek apa pun yang mungkin telah tertutup oleh grafik.
